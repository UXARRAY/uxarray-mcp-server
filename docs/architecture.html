Reading the swim lanes: Flow runs top → bottom, arrows crossing lanes are handoffs. Orange diamonds = routing decisions made automatically by the agent. Green boxes = compute runs on your local machine. Orange boxes = dispatched to a named HPC endpoint via Globus Compute — the file never leaves the cluster. Teal dashed boxes = steps that only activate under certain conditions (data_path provided, remote mode requested). Red dashed inside HPC boxes = automatic local fallback when the endpoint is unreachable. Small public surface: MCP clients call front-door tools such as run_analysis, plot_dataset, and diagnose_endpoint; implementation functions stay behind those dispatchers.
